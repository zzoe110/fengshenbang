# 烽审榜网站 - 部署指南

## 🎯 推荐：Vercel 免费部署（3分钟上线）

Vercel 提供：
- ✅ 全球CDN加速
- ✅ 自动HTTPS证书
- ✅ 自定义域名绑定
- ✅ Git 自动部署
- ✅ 免费额度足够中小企业站

### 部署步骤

#### 1️⃣ 注册 Vercel
- 访问 https://vercel.com
- 用 GitHub 账号登录（推荐）

#### 2️⃣ 推送代码到 GitHub
```bash
cd fengshenbang
git init
git add .
git commit -m "init: 烽审榜官网"
git branch -M main
git remote add origin https://github.com/你的用户名/fengshenbang.git
git push -u origin main
```

#### 3️⃣ Vercel 导入项目
1. Vercel 控制台 → "Add New..." → "Project"
2. 选择 `fengshenbang` 仓库
3. 框架选 "Other"，无构建命令
4. 点击 "Deploy"
5. ✅ 完成！会得到一个 `xxx.vercel.app` 域名

#### 4️⃣ 绑定自定义域名
1. Project Settings → Domains
2. 添加你的域名（如 `fengshenbang.com`）
3. 按提示到域名服务商配置 DNS（CNAME 记录）
4. 等待 SSL 证书签发（通常几分钟）

## 🇨🇳 国内部署（备案后）

### 腾讯云 COS 静态托管
1. 开通 COS 存储桶，开启「静态网站托管」
2. 把 `fengshenbang` 文件夹上传到存储桶根目录
3. 绑定自定义域名 + CDN 加速
4. SSL 证书可申请免费证书

### 阿里云 OSS 静态托管
1. 创建 OSS Bucket，开启静态网站托管
2. 上传文件
3. 绑定域名 + 阿里云CDN

## 🆘 常见问题

**Q: 后台数据存在哪里？会丢吗？**  
A: 演示版存浏览器 LocalStorage（清浏览器会丢）。生产环境务必接后端API。

**Q: 怎么备份数据？**  
A: 后台 → 站点设置 → 数据管理 → 导出全部数据，会下载一个 JSON 备份文件。

**Q: 怎么迁移到付费服务器？**  
A: 数据导出 → 新服务器部署 → 数据导入，即可完整迁移。

**Q: Logo 怎么换？**  
A: 把图片文件放到 `assets/images/`，然后后台 → 站点设置 → Logo URL 里填路径。
