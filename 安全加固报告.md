### 🛡️ 安全模块（security.js）能力

#### 1. 密码哈希 — PBKDF2
- 使用 Web Crypto API（浏览器原生加密）
- 10万次 SHA-256 迭代（暴力破解成本极高）
- 自动生成随机盐值
- 首次访问自动生成安全随机密码

#### 2. 登录保护
- **限流**：5次失败锁定15分钟
- **会话超时**：30分钟无操作自动登出
- 每次操作自动刷新会话时间
- 审计日志记录每次登录尝试（含时间戳、用户、UA）

#### 3. XSS 防护
- `escapeHTML()` — 转义 `& < > " ' /`
- `sanitizeHTML()` — 基于 DOM 的净化器
- 所有 Toast/通知使用 `textContent` 而非 `innerHTML`

#### 4. 审计日志
- 200条循环保留
- 记录：登录成功/失败、数据修改、配置变更、SEO更新
- 过滤查看：全部/登录/数据/配置/SEO
- 可清空（二次确认）

#### 5. 全局错误捕获
- `window.onerror` — JS 运行时错误
- `unhandledrejection` — Promise 异常
- 自动记录到审计日志

#### 6. CSP 内容安全策略
```
default-src 'self';
script-src 'self' 'unsafe-inline' https://fonts.googleapis.com;
style-src  'self' 'unsafe-inline' https://fonts.googleapis.com;
font-src   'self' https://fonts.gstatic.com;
img-src    'self' data:;
```

#### 7. 安全数据处理（演示版）
- **定时自动备份**：无（演示版特性，生产需接数据库）
- **数据完整性**：每次写入记录 `updatedBy` 字段
- **兼容旧数据**：`encrypted/plaintext` 双格式读取

### 🏗️ 仍需生产环境加固（需后端支持）
